LatentDenseGCN Fix Iteration 3 (current)
Objective
- Resolve persistent teacher collapse and improve Phase-1 teacher quality before further full-pipeline iterations.

Changes made
1) Model-level signal improvement (src/gcn_model.py)
- Added sample-conditioned structural node features from adjacency rows (A_hat) via struct_proj.
- Added learnable blend between structural features and static node embeddings (alpha gate).
- Rationale: static embeddings alone underuse per-sample topology and can collapse to majority-class behavior.

2) Training-level anti-collapse improvement (src/train_3class.py)
- Added class-weight mode: sqrt_inverse (less aggressive than inverse).
- Added anti-collapse regularizer on batch mean class probabilities.
- Kept label smoothing in CE loss.
- Set default teacher_class_weight_mode to sqrt_inverse in CLI path.

3) Runner defaults
- run_phase1_quality_3class.sh now uses --teacher_class_weight_mode sqrt_inverse.
